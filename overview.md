## Salesforce App Installer
It is a task to install a managed package into an org by providing org *credential* and managed package's *namespace* and *package version* number.

It allows you to install a Salesforce Appexchange App by providing its version number and its namespace plus the org credentials where you want to install that app.

It uses [Salesforce API for package install](https://developer.salesforce.com/docs/atlas.en-us.packagingGuide.meta/packagingGuide/packaging_api_introduction.htm)

### Note
* tested with ubuntu image
